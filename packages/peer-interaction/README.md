# Peer Interaction

Provides a number of abstractions and patterns for the communication between nodes in the XYO network.

## Install

Using yarn

```sh
  yarn add @xyo-network/peer-interaction
```

Using npm

```sh
  npm install @xyo-network/peer-interaction --save
```
