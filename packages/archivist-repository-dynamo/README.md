# Archivist Repository DynamoDB

Implements the archivist repository backed by DynamoDB

## Install

Using yarn

```sh
  yarn add @xyo-network/archivist-repository-dynamo
```

Using npm

```sh
  npm install @xyo-network/archivist-repository-dynamo --save
```
