# Diviner Archivist Client Graphql

A graphql client for communicating with Archivists

## Install

Using yarn

```sh
  yarn add @xyo-network/diviner-archivist-client
```

Using npm

```sh
  npm install @xyo-network/diviner-archivist-client --save
```
