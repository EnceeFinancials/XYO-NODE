# Transaction-pool

Aggregates transactions across different nodes in the network so that they can go into a block

## Install

Using yarn

```sh
  yarn add @xyo-network/transaction-pool
```

Using npm

```sh
  npm install @xyo-network/transaction-pool --save
```
