# Diviner Graphql API

Implements the the Diviner graphql API

## Install

Using yarn

```sh
  yarn add @xyo-network/app-diviner
```

Using npm

```sh
  npm install @xyo-network/app-diviner --save
```

Or install globally

```sh
  yarn global add @xyo-network/app-diviner
```

then to run

```sh
  xyo-diviner
```
