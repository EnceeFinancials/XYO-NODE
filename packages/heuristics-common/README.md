# Heuristics Common

A number of core heuristics used in the XYO protocol. In particular

- rssi
- latitude
- longitude
- gps
- time

## Install

Using yarn

```sh
  yarn add @xyo-network/heuristics-common
```

Using npm

```sh
  npm install @xyo-network/heuristics-common --save
```
