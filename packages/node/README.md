# Node

Implements a node life-cycle loop that can be used / extended to manage how a node in the XYO network behaves in different scenarios.

## Install

Using yarn

```sh
  yarn add @xyo-network/node
```

Using npm

```sh
  npm install @xyo-network/node --save
```
